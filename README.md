# Finance Dashboard App 💰📊

A full-stack MERN Finance Dashboard Application with Machine Learning Predictions.

This project is built using modern frontend and backend technologies to create a responsive finance analytics dashboard with charts, transactions, KPIs, and predictive insights.

---

## 🚀 Features

- Finance Dashboard UI  
- Interactive Charts & Analytics  
- Transactions Management  
- Product Performance Tracking  
- Machine Learning Predictions  
- Dark / Light Theme  
- Fully Responsive Design  
- Fast Performance  
- REST API Backend  
- Deployment Ready  

---

## 🛠️ Technologies Used

### Frontend

- React.js  
- TypeScript  
- Vite  
- Redux Toolkit  
- Redux Toolkit Query  
- Material UI (MUI)  
- Recharts  
- React Router DOM  

### Backend

- Node.js  
- Express.js  

### Database

- MongoDB  

### Deployment / Tools

- Docker  
- Vercel  
- Fly.io
